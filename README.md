# Nspray

This is a simple raytracer for the TI-Nspire (only really tested with the CX II) that uses [ndless](https://github.com/ndless-nspire/Ndless). By default, it renders a few spheres as well as a plane. A lot of the code was taken from [Ray Tracing in One Weekend](https://raytracing.github.io/books/RayTracingInOneWeekend.html) and rewritten in Rust.
